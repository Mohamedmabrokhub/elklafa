## إزاي ترفعه على GitHub Pages (خطوة بخطوة)

1. اعمل حساب على [github.com](https://github.com) لو معندكش.
2. اعمل Repository جديد (New repository)، سميه مثلاً `kalafa-app`، خليه **Public**.
3. ارفع كل الملفات اللي جوا الفولدر ده (`index.html`, `style.css`, `app.js`, `manifest.json`, `service-worker.js`, `.nojekyll`, وفولدر `assets`) لنفس الـ Repository:
   - أسهل طريقة: من صفحة الـ Repository اضغط **Add file → Upload files**، واسحب كل الملفات والفولدر، واعمل Commit.
4. روح لـ **Settings → Pages** في نفس الـ Repository.
5. تحت **Build and deployment** اختار Source: **Deploy from a branch**، وبعدين اختار Branch: `main` والفولدر `/ (root)`، واضغط **Save**.
6. استنى دقيقة أو اتنين، وهيديك لينك شبه ده:
   `https://<اسم-حسابك>.github.io/kalafa-app/`
7. افتح اللينك ده من موبايلك (Safari لو آيفون، Chrome لو أندرويد).
8. **مهم للشكل الاحترافي (Fullscreen زي تطبيق حقيقي):**
   - **آيفون:** من Safari اضغط زر المشاركة (Share) ⬆️ → **Add to Home Screen** → افتح الأيقونة من الشاشة الرئيسية (مش من المتصفح).
   - **أندرويد:** هيظهرلك اقتراح تلقائي "Add to Home screen" أو من قائمة المتصفح (⋮) اختار **Add to Home screen**.
   - بعد كده هتفتحه من الأيقونة على الشاشة الرئيسية وهيشتغل Fullscreen بدون شريط المتصفح، زي أي تطبيق عادي.

## تفعيل المزامنة بين الخمسة أشخاص (مهم جدًا!)

التطبيق شغال أساسًا محليًا (على كل موبايل لوحده). علشان الخمسة يشوفوا مواقف بعض، لازم تفعّلوا "المزامنة السحابية" المجانية دي (دقيقتين بس، مرة واحدة):

1. روح [jsonbin.io](https://jsonbin.io) واعمل حساب مجاني (Sign Up).
2. من الـ Dashboard اضغط **Create Bin**، واكتب جواه: `{"entries": []}` ثم Create.
3. هيديك **Bin ID** فوق الصفحة (انسخه).
4. من صفحة حسابك (الأيقونة فوق يمين) روح لـ **API Keys**، وانسخ الـ **X-MASTER-KEY**.
5. افتح تطبيق الكلافة → تبويب **إعدادات ⚙️** → حط الـ Bin ID والـ Master Key → فعّل السويتش → اضغط "حفظ واختبار الاتصال".
6. ابعت نفس الـ Bin ID والـ Master Key لباقي الأربعة، وكل واحد يحطهم في تبويب الإعدادات عنده ويفعّل السويتش.

بعد كده أي موقف أو تقييم بيتسجل عند أي حد، هيوصل لباقي الجروب تلقائيًا (بيعمل مزامنة كل شوية، وممكن تضغط على أيقونة ☁️ فوق عشان تحدّث يدويًا).

> ملحوظة: الـ Master Key ده بيدي صلاحية كتابة على الـ Bin بتاعك، متشاركوش غير جوا الجروب بتاعكم.

## المميزات

- تسجيل موقف غباء واحد في اليوم لكل شخص.
- تقييم مواقف الباقيين بمقياس غباء تفاعلي من 0 لـ 10.
- ترتيب أسبوعي فيه "ملك الغباء" مع تأثير احتفال (كونفيتي 🎉).
- سجل كامل لكل المواقف مع فلترة بالاسم.
- لوحة أدمن لمبروك: تصفير الأسبوع أو مسح كل البيانات.
- نسخة احتياطية (Export) للبيانات كملف JSON.
- يعمل Fullscreen زي تطبيق حقيقي بعد إضافته للشاشة الرئيسية (PWA).
